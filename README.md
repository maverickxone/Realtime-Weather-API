# Weather-Gemini-API

## 项目简介

本仓库用于存储我自主搭建的网站代码。
网站基于一个学生优惠半年有效期的阿里云服务器
其主要功能是根据IP进行天气查询和基本AI对话

网站接入了多种API接口，包括：

- 根据 IP 查询经纬度
- 天气基础信息查询 API
- 集成 Gemini 的高级 API

旨在为各类应用或前端项目提供便捷的数据接口服务，无论是简单的地理位置查询，还是更精准和高级的天气与AI能力对接，都可通过本项目获得支持。

## 使用方式（后期补充）

1. 克隆本仓库到你的服务器或本地环境；
2. 配置相应的API密钥与环境变量；
3. 启动服务，根据需求调用对应API接口。

## 适用场景

- Web开发需要天气或位置服务
- 学习基础API调用
