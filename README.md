# Realtime-Weather-API

## 项目简介

本仓库用于存储我自主搭建的网站代码。
网站基于一个学生优惠半年有效期的阿里云服务器
其主要功能是根据IP进行天气查询，异步加载，AI回复的流式输出

网站接入了多种API接口，包括：

- 高德API：根据 IP 查询经纬度
- 和风天气API：根据经纬度查询天气信息
- QwenAPI: 集成 Qwen3-8B给出个性化回复

本项目主要是展示个人网站成果，当然其他人也可从中学习基础的API调用及其他技能。
请注意，我在app.py中隐去了所有API-key，需要者应该自行前往官网获取。

## 使用方式

1. 克隆本仓库到你的服务器或本地环境；
2. 配置相应的API密钥与环境变量；
3. 启动服务，根据需求调用对应API接口。

## 适用场景

- Web开发需要天气或位置服务
- 学习基础API调用
